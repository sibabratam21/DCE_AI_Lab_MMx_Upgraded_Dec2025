Test Recalibration

Open the browser console and navigate to the app at http://localhost:5174

Then follow these steps:

  1. Go to the Modeling tab
  2. Click "Recalibrate"
  3. Select only 2 channels in the wizard
  4. Complete the wizard and click "Start Training"
  5. Watch the console for these debug messages:

Common Issues to Check:

  • If no models generated: Check "[TrainingAPI] Generated models: { count: 0 }"
  • If models not reaching leaderboard: Check "[App.tsx] onModelsUpdated callback received"
